## **주요 기능 소개**

### **릴스(숏폼) 기능**

<p>
<img alt="릴스" src="https://github.com/DevReelsTeam/DevReels/assets/121999075/01aa92ed-8697-495b-ba8a-046c8bc52c4b" width="350>
</p>
  
- 관심있는 상대방을 팔로우할 수 있습니다.
- 좋아요를 눌러 관심을 표시할 수 있습니다.
- 기능에 해당하는 깃허브로 이동할 수 있습니다.



### **기능에 대해 의견(댓글)남기기**

<p>
